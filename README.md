# 🎥 Cam Overlay — Diablo / Cyber Edition

## 🔥 Présentation
**Cam Overlay — Diablo / Cyber Edition** est un overlay caméra avancé pour stream (Twitch / OBS), conçu pour habiller ta caméra avec un cadre dynamique, des effets visuels réactifs et un système de stats + réseaux intégré.

Deux styles disponibles :
- ⚔️ **Diablo** (dark fantasy)
- ⚡ **Cyber** (néon futuriste)

## ⚙️ Fonctionnement

Le projet repose sur 4 fichiers :

- `CAM_HTML.html` → Structure overlay  
- `CAM_CSS.css` → Design + animations  
- `CAM_JS.js` → Logique dynamique  
- `CAM_FIELDS.json` → Configuration complète

## 🚀 Installation

### 1. Aller sur StreamElements
- Connecte-toi à ton compte
- Va dans **Overlays**

---

### 2. Créer un overlay
- Clique sur **New Overlay**
- Choisis la résolution
- Ouvre l’éditeur

---

### 3. Ajouter un widget custom
- Clique sur **Add Widget**
- Sélectionne **Custom Widget**

---

### 4. Importer les fichiers

Dans le widget :

- Onglet **HTML** → colle `CAM_HTML.html`  
- Onglet **CSS** → colle `CAM_CSS.css`  
- Onglet **JS** → colle `CAM_JS.js`  
- Onglet **Fields** → colle `CAM_FIELDS.json`  

---

### 5. Finaliser
- Clique sur **Done**
- Sauvegarde l’overlay

---

### 6. Ajouter dans OBS
- Ajoute une **source navigateur**
- Colle l’URL de l’overlay
